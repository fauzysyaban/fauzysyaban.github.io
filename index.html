Smart Parking Monitoring System

Problem & Solution

Mencari slot parkir kosong di Gedung Kuliah Umum sering membuang waktu. Alat ini dibuat menggunakan sensor deteksi objek pada setiap slot yang kemudian mengirimkan status keterisian secara nirkabel ke pangkalan data pusat, sehingga pengguna bisa mengecek ketersediaan parkir via aplikasi.

Spesifikasi Perangkat

Menggunakan NodeMCU/ESP8266, Sensor Ultrasonik/Inframerah, dan Protokol MQTT untuk komunikasi data yang cepat dan hemat daya.

LoRa GPS Tracker & Emergency Signal

Problem & Solution

Hilangnya sinyal seluler di area pegunungan membuat koordinasi penyelamatan pendaki terhambat. Alat ini memadukan modul GPS untuk membaca titik koordinat presisi dan memancarkannya menggunakan sinyal radio LoRa (Long Range) ke pos pemantau terjauh.

Spesifikasi Perangkat

Menggunakan ESP32 Development Board, Modul GPS NEO-6M, Modul LoRa RFM95W, dan optimasi konsumsi baterai sirkuit dalam mode deep-sleep.
